PCF metadata extractor

Documentation

Help

CLI equivalent: pcf-inventory-extract <org> [-o file] [-d] (requires cf login on the host).

Options

  • org — required Cloud Foundry organization name
  • -o / output file — optional path for the CSV; default is pcfusage_<org>_YYYYMMDDHHMMSS.csv in the server process working directory
  • -d / debug — extra diagnostics to server stderr

Web UI

The web form asks for the CF API URL (same as cf login -a), username, password, and org. The server obtains a bearer token from UAA and runs the extract; it does not rely on a cf CLI session in your home directory. SSO-only foundations may reject password login.
