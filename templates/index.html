PCF metadata extractor

Cloud Foundry org inventory (CSV)

Collects org, space, app, and process data from the Cloud Foundry v3 API and writes one CSV you can use for migration planning, audits, and reporting.

Sign in with your Cloud Foundry API URL and account. The server uses a one-time UAA token for this request; it does not store your password. Use HTTPS in front of this app in production.
